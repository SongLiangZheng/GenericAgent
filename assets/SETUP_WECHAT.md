## 快速开始

### 步骤 1：运行登录脚本

微信机器人采用**扫码登录**方式，无需创建应用或申请权限。

直接运行以下命令：

```bash
cd D:\workspace\me\GenericAgent
python frontends/wechatapp.py
```

### 步骤 2：扫码登录

1. 脚本会自动生成二维码图片并在浏览器中打开
2. 打开微信，切换到**登录的微信账号**（不是文件传输助手）
3. 使用微信扫一扫功能扫描屏幕上的二维码
4. 在手机上确认登录

### 步骤 3：验证成功

登录成功后，终端会显示：

```
[QR登录] 成功! bot_id=xxxxx
```

此时，向你的微信**文件传输助手**或任意群聊发送消息，机器人即可响应。

## 项目配置

### 配置 LLM

确保 `mykey.py` 中已配置 LLM API 密钥：

```python
# 示例：Claude API
claude_config = {
    'apikey': 'sk-ant-xxxxx',
    'apibase': 'https://api.anthropic.com',
    'model': 'claude-sonnet-4-20250514'
}
```

### Token 存储

登录成功后，Token 会自动保存到：

```
C:\Users\你的用户名\.wxbot\token.json
```

**重要提示**：
- Token 有效期较长，通常无需重复登录
- 删除此文件将重新触发扫码登录
- 请勿将此文件分享给他人

## 常见问题

### 1. 二维码无法加载？

**原因**：网络问题或微信服务波动

**解决**：
- 检查网络连接
- 稍后重试：重新运行脚本

### 2. 登录后没反应？

**原因**：可能是消息监听延迟

**解决**：
- 检查终端是否有报错
- 向**文件传输助手**发送消息测试
- 确保手机微信已登录

### 3. 如何控制谁能使用机器人？

**方法 1**：修改 `wechatapp.py` 中的用户过滤逻辑
**方法 2**：仅在特定群聊中启用机器人

### 4. Token 失效怎么办？

**解决**：
- 删除 `C:\Users\你的用户名\.wxbot\token.json`
- 重新运行脚本扫码登录

### 5. 支持哪些功能？

- **文本消息**：收发文本
- **文件传输**：接收用户发送的文件
- **图片识别**：接收图片并进行 OCR
- **执行系统命令**：通过对话控制电脑
- **Agent 推理**：调用 LLM 进行复杂任务规划

## 进阶配置

### 自定义欢迎语

编辑 `wechatapp.py`，修改 `on_message` 函数：

```python
if text.startswith('你好'):
    return '你好！我是你的 AI 助手，有什么可以帮你的吗？'
```

### 添加权限控制

在 `on_message` 函数开头添加：

```python
ALLOWED_USERS = ['your_open_id_1', 'your_open_id_2']
if user_id not in ALLOWED_USERS:
    return '抱歉，你没有权限使用此机器人。'
```

## 安全提示

1. **Token 安全**：请勿将 `token.json` 上传到公共仓库
2. **权限控制**：建议限制机器人仅响应特定用户或群聊
3. **命令执行**：机器人具有系统命令执行权限，请谨慎使用
4. **网络隔离**：建议在受信任的网络环境中运行
